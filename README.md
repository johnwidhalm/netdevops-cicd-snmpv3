## Installation Steps
* install jenkins container
* install ansible or ansible plugin
* install ansible lint (apt install python3-pip, pip3 install ansible-lint)
* install slack plugin

## Pipeline

* branch created.
* change made to branch locally

* change pushed
  * STEP1 : Lint
  * STEP2 : Deploy to test
  * STEP3 : Test test env

..
